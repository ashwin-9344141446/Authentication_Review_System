Authentication & Feedback System;
A frontend web application developed as part of the VIPS TECH Certification Task.
This project demonstrates a clean and responsive User Authentication flow (Sign Up & Sign In) along with a Feedback and Review system, built using core web technologies.

Project Overviewm;
This project focuses on creating essential UI components commonly used in real-world web applications:
User Registration (Sign Up)
User Login (Sign In)
Star-based Feedback & Review submission
The goal of this task was to apply frontend fundamentals, follow structured design practices, and build user-friendly interfaces.

Technologies Used;
HTML5 – Semantic structure
CSS3 – Styling and layout
JavaScript – Basic interactivity and form behavior
Google Fonts & Icons – UI enhancement

Features;
